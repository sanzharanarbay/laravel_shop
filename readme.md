## How to install app?


 - git clone https://github.com/sanzharanarbay/laravel_shop.git
 - composer install
 - cp .env.example .env
 - Configure DB settings on .env file
 - Create Mysql DB on your Mysql server and import .sql file from DBSQL folder
 - php artisan key:generate
 - php artisan config:clear
 - php artisan serve
